# Ejecutar en Dev

1. Clonar el repositorio
2. Instalar dependencias con `npm install`
3. Clonar el `.env.template` y renombrar a `.env`
3. Levantar la base de datos `docker compose up -d`
4. Ejecutar proyecto `npm run start:dev`
